# Week 6 - Constructors and Data Encapsulation

## Overview

This week focuses on private data members, constructor overloading, and secure data handling.

## Learning Objectives

* Work with private members
* Use parameterized constructors
* Improve encapsulation
* Display object data through functions

## Experiments

### Experiment 1: Student Information System

Store:

* Roll Number
* First Name
* Last Name
* Class
* Marks
* Grade

Implement:

* Default Constructor
* Parameterized Constructor
* Data Display Function

## Concepts Covered

* Encapsulation
* Constructors
* Private Members
* Object-Oriented Design

## Language Used

C++

## Outcome

Students learned how to protect and initialize object data using modern OOP practices.
